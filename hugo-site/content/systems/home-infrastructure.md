A self-hosted ecosystem connecting creative tools, media archives, and mobile workflows.

## Hardware

- **UGREEN DXP4800 NAS** — 3×8TB RAID 5 + 4TB SSD cache
- **2012 Mac mini** — OpenCore Legacy Patcher running Monterey
- **Supernote** — E-ink tablet for portable reading and annotation

## Services

- **Plex Media Server** — Docker container for music streaming via Plexamp
- **Tailscale** — Mesh VPN for secure remote access
- **WebDAV** — Supernote ebook library access (1000+ volumes)
- **Real-Debrid** — Streaming integration via Zurg/rclone

## In Progress

- Readarr for ebook management automation
- Beets for music library organization
- CalDAV calendar sync exploration
